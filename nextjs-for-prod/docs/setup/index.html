Setup & Installation

This guide will help you set up the Next.js for Production project on your local machine.

Prerequisites​

Before you begin, ensure you have the following installed:

  • Node.js (v18.0 or higher) - Download
  • pnpm (recommended package manager) - Install pnpm
  • Git - Download
  • Docker (optional, for containerization) - Download

Installation Steps​

1. Clone the Repository​

git clone https://github.com/tamnk74/nextjs-for-prod.git
cd nextjs-for-prod

2. Install Dependencies​

Using pnpm (recommended):

pnpm install

Or using npm:

npm install

Or using yarn:

yarn install

3. Environment Configuration​

Create your environment file:

cp .env.example .env.local

Edit .env.local with your configuration:

# Next.js Configuration
NEXT_PUBLIC_APP_URL=http://localhost:3000

# Supabase Configuration
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key

# Elasticsearch Configuration (Optional)
ELASTICSEARCH_URL=http://localhost:9200
ELASTICSEARCH_USERNAME=elastic
ELASTICSEARCH_PASSWORD=your_password

# Turnstile Configuration (Optional)
NEXT_PUBLIC_TURNSTILE_SITE_KEY=your_turnstile_site_key
TURNSTILE_SECRET_KEY=your_turnstile_secret_key

# Other Configuration
NODE_ENV=development

4. Database Setup​

Option A: Using Supabase Cloud​

  1. Create a new project at supabase.com
  2. Copy your project URL and anon key to .env.local
  3. Run the database migrations:
# Install Supabase CLI
npm install -g @supabase/cli

# Login to Supabase
supabase login

# Link your project
supabase link --project-ref your-project-id

# Push database schema
supabase db push

Option B: Local Supabase Development​

# Start local Supabase
supabase start

# Apply migrations
supabase db reset

5. Run the Development Server​

pnpm dev

Open http://localhost:3000 in your browser.

Development Tools Setup​

ESLint and Prettier​

These are pre-configured. To run manually:

# Lint code
pnpm lint

# Fix linting issues
pnpm lint:fix

# Format code
pnpm format

Git Hooks​

Husky is configured for pre-commit hooks:

# Install git hooks
pnpm prepare

TypeScript​

Check types:

pnpm type-check

Optional Services Setup​

Elasticsearch (for Library Search)​

Using Docker:​

docker run -d \
  --name elasticsearch \
  -p 9200:9200 \
  -p 9300:9300 \
  -e "discovery.type=single-node" \
  -e "xpack.security.enabled=false" \
  elasticsearch:8.8.0

Initialize search data:​

pnpm run init-search

Redis (for Caching)​

Using Docker:​

docker run -d \
  --name redis \
  -p 6379:6379 \
  redis:alpine

Docker Setup (Optional)​

Development with Docker​

# Build and run with Docker Compose
docker-compose up -d

# Or build manually
docker build -t nextjs-for-prod .
docker run -p 3000:3000 nextjs-for-prod

Production Docker Build​

# Build production image
docker build -t nextjs-for-prod:prod --target production .

# Run production container
docker run -p 3000:3000 nextjs-for-prod:prod

Verification​

After setup, verify everything works:

1. Application Health​

Visit http://localhost:3000 - should show the homepage

2. API Health​

Visit http://localhost:3000/api/health - should return status OK

3. Authentication​

Try creating an account and logging in

4. Database Connection​

Check if data loads correctly in the application

Troubleshooting​

Common Issues​

Port Already in Use​

# Kill process on port 3000
npx kill-port 3000

# Or use different port
pnpm dev -- -p 3001

Database Connection Issues​

  • Verify Supabase credentials in .env.local
  • Check if Supabase project is active
  • Ensure database migrations are applied

Elasticsearch Connection Issues​

  • Verify Elasticsearch is running: curl http://localhost:9200
  • Check credentials and URL in .env.local
  • Restart Elasticsearch service

Node Version Issues​

# Check Node version
node --version

# Use nvm to switch versions
nvm use 18

Getting Help​

If you encounter issues:

  1. Check the GitHub Issues
  2. Review the common troubleshooting section above
  3. Create a new issue with detailed information
