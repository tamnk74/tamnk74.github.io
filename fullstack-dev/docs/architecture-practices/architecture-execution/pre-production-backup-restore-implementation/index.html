Pre-Production Environments Backup/Restore Strategy Implementation Guide

This guide provides step-by-step instructions for implementing comprehensive backup and restore strategies for development, staging, and testing environments in your NestJS/Next.js application ecosystem.

Overview​

Pre-production environments require specialized backup and restore strategies that balance data protection with development agility. Unlike production, these environments need frequent refreshes, test data management, and rapid environment provisioning capabilities.

Success Criteria​

By the end of this implementation, you should have:

  • ✅ Automated backup systems for all pre-production environments
  • ✅ Environment refresh capabilities for consistent testing
  • ✅ Test data management and privacy compliance
  • ✅ Rapid environment provisioning from backups
  • ✅ Data synchronization between environments
  • ✅ Monitoring and alerting for backup health

    ## Data Categories in Pre-Production

    ### Sensitive Data (PII/PHI)

    - **Production Mirror**: Real user data (masked/anonymized)
    - **Synthetic Data**: Generated test data resembling production
    - **Compliance**: GDPR, HIPAA, SOX requirements

    ### Application Data

    - **Configuration**: Environment-specific settings
    - **Application State**: User sessions, cache data
    - **File Uploads**: Test documents, images, media files

    ### System Data

    - **Database Schemas**: Table structures, indexes, constraints
    - **Application Code**: Deployed versions and configurations
    - **Infrastructure Config**: Kubernetes manifests, Terraform state

Phase 2: Database Backup Implementation​

Step 2.1: Automated Database Backups​

Duration: 3-4 days

Actions:

  1. Database Backup Service Implementation

    // src/backup/pre-production-backup.service.ts

    import { Injectable, Logger } from '@nestjs/common';
    import { ConfigService } from '@nestjs/config';
    import { Cron } from '@nestjs/schedule';
    import { InjectRepository } from '@nestjs/typeorm';
    import { Repository, DataSource } from 'typeorm';
    import { Storage } from '@google-cloud/storage';
    import { exec } from 'child_process';
    import { promisify } from 'util';
    import * as fs from 'fs/promises';
    import * as path from 'path';
    import * as zlib from 'zlib';

    const execAsync = promisify(exec);

    export interface BackupJob {
      id: string;
      environment: string;
      type: 'database' | 'files' | 'full';
      status: 'pending' | 'running' | 'completed' | 'failed';
      startTime: Date;
      endTime?: Date;
      size?: number;
      location?: string;
      metadata?: Record<string, any>;
    }

    @Injectable()
    export class PreProductionBackupService {
      private readonly logger = new Logger(PreProductionBackupService.name);
      private readonly storage = new Storage();

      constructor(private readonly configService: ConfigService, private readonly dataSource: DataSource) {}

      @Cron('0 2 * * *') // Daily at 2 AM
      async performDevelopmentBackup(): Promise<void> {
        await this.performEnvironmentBackup('development');
      }

      @Cron('0 */1 * * *') // Hourly
      async performStagingBackup(): Promise<void> {
        await this.performEnvironmentBackup('staging');
      }

      @Cron('0 */6 * * *') // Every 6 hours
      async performUATBackup(): Promise<void> {
        await this.performEnvironmentBackup('uat');
      }

      async performEnvironmentBackup(environment: string): Promise<BackupJob> {
        const job: BackupJob = {
          id: `backup-${environment}-${Date.now()}`,
          environment,
          type: 'full',
          status: 'pending',
          startTime: new Date(),
        };

        try {
          this.logger.log(`Starting backup for ${environment} environment`);
          job.status = 'running';

          // 1. Create database backup
          const dbBackupPath = await this.createDatabaseBackup(environment, job.id);

          // 2. Create file system backup
          const fileBackupPath = await this.createFileSystemBackup(environment, job.id);

          // 3. Upload to cloud storage
          const cloudLocation = await this.uploadBackupToCloud(environment, job.id, [dbBackupPath, fileBackupPath]);

          // 4. Clean up local files
          await this.cleanupLocalBackups([dbBackupPath, fileBackupPath]);

          // 5. Update job status
          job.status = 'completed';
          job.endTime = new Date();
          job.location = cloudLocation;

          this.logger.log(`Backup completed for ${environment}: ${cloudLocation}`);

          // 6. Send notification
          await this.sendBackupNotification(job);

          return job;
        } catch (error) {
          job.status = 'failed';
          job.endTime = new Date();
          this.logger.error(`Backup failed for ${environment}:`, error.stack);

          await this.sendBackupNotification(job, error);
          throw error;
        }
      }

      private async createDatabaseBackup(environment: string, jobId: string): Promise<string> {
        const timestamp = new Date().toISOString().replace(/[:.]/g, '-');
        const filename = `${environment}-db-backup-${timestamp}.sql`;
        const backupPath = path.join('/tmp', filename);

        const dbConfig = this.configService.get(`database.${environment}`);

        // Create compressed database dump
        const pgDumpCommand = [
          'pg_dump',
          `--host=${dbConfig.host}`,
          `--port=${dbConfig.port}`,
          `--username=${dbConfig.username}`,
          `--dbname=${dbConfig.database}`,
          '--format=custom',
          '--compress=9',
          '--no-owner',
          '--no-privileges',
          `--file=${backupPath}`,
        ].join(' ');

        this.logger.log(`Creating database backup: ${filename}`);

        try {
          await execAsync(pgDumpCommand, {
            env: { ...process.env, PGPASSWORD: dbConfig.password },
          });

          const stats = await fs.stat(backupPath);
          this.logger.log(`Database backup created: ${filename} (${stats.size} bytes)`);

          return backupPath;
        } catch (error) {
          this.logger.error(`Database backup failed: ${error.message}`);
          throw error;
        }
      }

      private async createFileSystemBackup(environment: string, jobId: string): Promise<string> {
        const timestamp = new Date().toISOString().replace(/[:.]/g, '-');
        const filename = `${environment}-files-backup-${timestamp}.tar.gz`;
        const backupPath = path.join('/tmp', filename);

        const dataDirectory = this.configService.get(`storage.${environment}.dataPath`);

        if (!dataDirectory) {
          this.logger.warn(`No data directory configured for ${environment}, skipping file backup`);
          return null;
        }

        const tarCommand = [
          'tar',
          '-czf',
          backupPath,
          '-C',
          path.dirname(dataDirectory),
          path.basename(dataDirectory),
        ].join(' ');

        try {
          this.logger.log(`Creating file system backup: ${filename}`);
          await execAsync(tarCommand);

          const stats = await fs.stat(backupPath);
          this.logger.log(`File system backup created: ${filename} (${stats.size} bytes)`);

          return backupPath;
        } catch (error) {
          this.logger.error(`File system backup failed: ${error.message}`);
          throw error;
        }
      }

      private async uploadBackupToCloud(environment: string, jobId: string, backupPaths: string[]): Promise<string> {
        const bucketName = this.configService.get('backup.storage.bucket');
        const bucket = this.storage.bucket(bucketName);

        const uploadPromises = backupPaths
          .filter((path) => path !== null)
          .map(async (localPath) => {
            const filename = path.basename(localPath);
            const cloudPath = `pre-production/${environment}/${jobId}/${filename}`;
            const file = bucket.file(cloudPath);

            this.logger.log(`Uploading ${filename} to ${cloudPath}`);

            await bucket.upload(localPath, {
              destination: cloudPath,
              metadata: {
                metadata: {
                  environment,
                  jobId,
                  timestamp: new Date().toISOString(),
                  type: filename.includes('db-backup') ? 'database' : 'files',
                },
              },
            });

            return cloudPath;
          });

        const uploadedPaths = await Promise.all(uploadPromises);
        return `gs://${bucketName}/pre-production/${environment}/${jobId}/`;
      }

      private async cleanupLocalBackups(backupPaths: string[]): Promise<void> {
        const cleanupPromises = backupPaths
          .filter((path) => path !== null)
          .map(async (backupPath) => {
            try {
              await fs.unlink(backupPath);
              this.logger.log(`Cleaned up local backup: ${backupPath}`);
            } catch (error) {
              this.logger.warn(`Failed to cleanup local backup ${backupPath}:`, error.message);
            }
          });

        await Promise.all(cleanupPromises);
      }

      private async sendBackupNotification(job: BackupJob, error?: Error): Promise<void> {
        const webhookUrl = this.configService.get('notifications.slack.webhookUrl');

        if (!webhookUrl) return;

        const color = job.status === 'completed' ? 'good' : 'danger';
        const message = {
          attachments: [
            {
              color,
              title: `Pre-Production Backup ${job.status === 'completed' ? 'Completed' : 'Failed'}`,
              fields: [
                { title: 'Environment', value: job.environment, short: true },
                { title: 'Job ID', value: job.id, short: true },
                { title: 'Duration', value: this.formatDuration(job.startTime, job.endTime), short: true },
                { title: 'Status', value: job.status, short: true },
              ],
              ...(error && { text: `Error: ${error.message}` }),
              ...(job.location && { text: `Backup location: ${job.location}` }),
            },
          ],
        };

        try {
          const response = await fetch(webhookUrl, {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify(message),
          });

          if (!response.ok) {
            throw new Error(`Slack notification failed: ${response.statusText}`);
          }
        } catch (notificationError) {
          this.logger.error('Failed to send backup notification:', notificationError.message);
        }
      }

      private formatDuration(start: Date, end?: Date): string {
        if (!end) return 'In progress';

        const durationMs = end.getTime() - start.getTime();
        const minutes = Math.floor(durationMs / 60000);
        const seconds = Math.floor((durationMs % 60000) / 1000);

        return `${minutes}m ${seconds}s`;
      }
    }

Step 2.2: Environment Refresh and Data Synchronization​

Duration: 2-3 days

Actions:

  1. Environment Refresh Service

    // src/backup/environment-refresh.service.ts

    import { Injectable, Logger } from '@nestjs/common';
    import { ConfigService } from '@nestjs/config';
    import { Storage } from '@google-cloud/storage';
    import { exec } from 'child_process';
    import { promisify } from 'util';
    import * as fs from 'fs/promises';
    import * as path from 'path';

    const execAsync = promisify(exec);

    export interface RefreshJob {
      id: string;
      sourceEnvironment: string;
      targetEnvironment: string;
      type: 'full' | 'data-only' | 'schema-only';
      status: 'pending' | 'running' | 'completed' | 'failed';
      startTime: Date;
      endTime?: Date;
      options?: {
        maskSensitiveData?: boolean;
        preserveUsers?: boolean;
        preserveConfigs?: boolean;
      };
    }

    @Injectable()
    export class EnvironmentRefreshService {
      private readonly logger = new Logger(EnvironmentRefreshService.name);
      private readonly storage = new Storage();

      constructor(private readonly configService: ConfigService) {}

      async refreshEnvironmentFromProduction(
        targetEnvironment: string,
        options: RefreshJob['options'] = {},
      ): Promise<RefreshJob> {
        const job: RefreshJob = {
          id: `refresh-${targetEnvironment}-${Date.now()}`,
          sourceEnvironment: 'production',
          targetEnvironment,
          type: 'full',
          status: 'pending',
          startTime: new Date(),
          options,
        };

        try {
          this.logger.log(`Starting environment refresh: production -> ${targetEnvironment}`);
          job.status = 'running';

          // 1. Create fresh production backup
          const prodBackupPath = await this.createProductionSnapshot();

          // 2. Download and prepare backup for target environment
          const processedBackupPath = await this.processBackupForEnvironment(prodBackupPath, targetEnvironment, options);

          // 3. Stop target environment services
          await this.stopEnvironmentServices(targetEnvironment);

          // 4. Restore database
          await this.restoreDatabase(targetEnvironment, processedBackupPath);

          // 5. Restore file system
          await this.restoreFileSystem(targetEnvironment, processedBackupPath);

          // 6. Update environment-specific configurations
          await this.updateEnvironmentConfigurations(targetEnvironment);

          // 7. Start services and verify
          await this.startEnvironmentServices(targetEnvironment);
          await this.verifyEnvironmentHealth(targetEnvironment);

          job.status = 'completed';
          job.endTime = new Date();

          this.logger.log(`Environment refresh completed: ${job.id}`);
          await this.sendRefreshNotification(job);

          return job;
        } catch (error) {
          job.status = 'failed';
          job.endTime = new Date();
          this.logger.error(`Environment refresh failed:`, error.stack);

          await this.sendRefreshNotification(job, error);
          throw error;
        }
      }

      async refreshEnvironmentFromBackup(
        targetEnvironment: string,
        backupId: string,
        options: RefreshJob['options'] = {},
      ): Promise<RefreshJob> {
        const job: RefreshJob = {
          id: `restore-${targetEnvironment}-${Date.now()}`,
          sourceEnvironment: 'backup',
          targetEnvironment,
          type: 'full',
          status: 'pending',
          startTime: new Date(),
          options,
        };

        try {
          this.logger.log(`Starting environment restore: ${backupId} -> ${targetEnvironment}`);
          job.status = 'running';

          // 1. Download backup from cloud storage
          const backupPath = await this.downloadBackup(backupId, targetEnvironment);

          // 2. Stop target environment services
          await this.stopEnvironmentServices(targetEnvironment);

          // 3. Restore from backup
          await this.restoreFromBackup(targetEnvironment, backupPath);

          // 4. Start services and verify
          await this.startEnvironmentServices(targetEnvironment);
          await this.verifyEnvironmentHealth(targetEnvironment);

          job.status = 'completed';
          job.endTime = new Date();

          this.logger.log(`Environment restore completed: ${job.id}`);
          await this.sendRefreshNotification(job);

          return job;
        } catch (error) {
          job.status = 'failed';
          job.endTime = new Date();
          this.logger.error(`Environment restore failed:`, error.stack);

          await this.sendRefreshNotification(job, error);
          throw error;
        }
      }

      private async processBackupForEnvironment(
        backupPath: string,
        targetEnvironment: string,
        options: RefreshJob['options'],
      ): Promise<string> {
        const processedPath = `${backupPath}.processed`;

        if (options.maskSensitiveData) {
          await this.maskSensitiveData(backupPath, processedPath);
        } else {
          await fs.copyFile(backupPath, processedPath);
        }

        if (options.preserveUsers) {
          await this.preserveEnvironmentUsers(processedPath, targetEnvironment);
        }

        if (options.preserveConfigs) {
          await this.preserveEnvironmentConfigs(processedPath, targetEnvironment);
        }

        return processedPath;
      }

      private async maskSensitiveData(sourcePath: string, targetPath: string): Promise<void> {
        this.logger.log('Masking sensitive data in backup');

        // Create a temporary SQL file with data masking commands
        const maskingScript = `
          -- Mask email addresses
          UPDATE users SET email = CONCAT('user_', id, '@example.com');
          
          -- Mask phone numbers
          UPDATE users SET phone = CONCAT('555-0', LPAD(id::text, 3, '0'));
          
          -- Mask names
          UPDATE users SET 
            first_name = CONCAT('User', id),
            last_name = CONCAT('Test', id);
          
          -- Mask addresses
          UPDATE addresses SET 
            street = CONCAT(id, ' Test Street'),
            city = 'Test City',
            postal_code = '12345';
          
          -- Remove sensitive logs
          DELETE FROM audit_logs WHERE created_at < NOW() - INTERVAL '30 days';
          
          -- Reset passwords to a default test value
          UPDATE users SET password_hash = '$2b$10$test.hash.for.development.environment';
        `;

        const maskingScriptPath = '/tmp/masking-script.sql';
        await fs.writeFile(maskingScriptPath, maskingScript);

        // Apply masking to the backup
        const dbConfig = this.configService.get('database.production');
        const maskCommand = [
          'pg_restore',
          '--clean',
          '--if-exists',
          '--no-owner',
          '--no-privileges',
          '--single-transaction',
          `--host=${dbConfig.host}`,
          `--port=${dbConfig.port}`,
          `--username=${dbConfig.username}`,
          `--dbname=temp_masking_db`,
          sourcePath,
          '&&',
          'psql',
          `--host=${dbConfig.host}`,
          `--port=${dbConfig.port}`,
          `--username=${dbConfig.username}`,
          `--dbname=temp_masking_db`,
          '--file',
          maskingScriptPath,
          '&&',
          'pg_dump',
          `--host=${dbConfig.host}`,
          `--port=${dbConfig.port}`,
          `--username=${dbConfig.username}`,
          `--dbname=temp_masking_db`,
          '--format=custom',
          '--no-owner',
          '--no-privileges',
          `--file=${targetPath}`,
        ].join(' ');

        await execAsync(maskCommand, {
          env: { ...process.env, PGPASSWORD: dbConfig.password },
        });

        // Cleanup
        await fs.unlink(maskingScriptPath);
      }

      private async stopEnvironmentServices(environment: string): Promise<void> {
        this.logger.log(`Stopping services for ${environment} environment`);

        const namespace = `${environment}-namespace`;
        const stopCommand = `kubectl scale deployment --all --replicas=0 -n ${namespace}`;

        try {
          await execAsync(stopCommand);

          // Wait for pods to terminate
          const waitCommand = `kubectl wait --for=delete pod --all -n ${namespace} --timeout=300s`;
          await execAsync(waitCommand);

          this.logger.log(`Services stopped for ${environment}`);
        } catch (error) {
          this.logger.error(`Failed to stop services for ${environment}:`, error.message);
          throw error;
        }
      }

      private async startEnvironmentServices(environment: string): Promise<void> {
        this.logger.log(`Starting services for ${environment} environment`);

        const namespace = `${environment}-namespace`;
        const startCommand = `kubectl scale deployment --all --replicas=1 -n ${namespace}`;

        try {
          await execAsync(startCommand);

          // Wait for pods to be ready
          const waitCommand = `kubectl wait --for=condition=ready pod --all -n ${namespace} --timeout=600s`;
          await execAsync(waitCommand);

          this.logger.log(`Services started for ${environment}`);
        } catch (error) {
          this.logger.error(`Failed to start services for ${environment}:`, error.message);
          throw error;
        }
      }

      private async verifyEnvironmentHealth(environment: string): Promise<void> {
        this.logger.log(`Verifying health for ${environment} environment`);

        const healthCheckUrl = this.configService.get(`environments.${environment}.healthCheckUrl`);

        if (!healthCheckUrl) {
          this.logger.warn(`No health check URL configured for ${environment}`);
          return;
        }

        let attempts = 0;
        const maxAttempts = 10;
        const delay = 30000; // 30 seconds

        while (attempts < maxAttempts) {
          try {
            const response = await fetch(healthCheckUrl);

            if (response.ok) {
              this.logger.log(`Environment ${environment} is healthy`);
              return;
            }

            this.logger.warn(`Health check failed for ${environment}: ${response.status}`);
          } catch (error) {
            this.logger.warn(`Health check attempt ${attempts + 1} failed:`, error.message);
          }

          attempts++;
          if (attempts < maxAttempts) {
            await new Promise((resolve) => setTimeout(resolve, delay));
          }
        }

        throw new Error(`Environment ${environment} failed health checks after ${maxAttempts} attempts`);
      }

Phase 4: Monitoring and Automation​

Step 4.1: Backup Monitoring and Alerting​

Duration: 2-3 days

Actions:

  1. Backup Monitoring Service

    // src/monitoring/backup-monitoring.service.ts

    import { Injectable, Logger } from '@nestjs/common';
    import { Cron } from '@nestjs/schedule';
    import { ConfigService } from '@nestjs/config';
    import { Storage } from '@google-cloud/storage';

    export interface BackupHealthReport {
      environment: string;
      lastBackupTime: Date;
      backupStatus: 'healthy' | 'warning' | 'critical';
      backupSize: number;
      retentionCompliance: boolean;
      issues: string[];
    }

    @Injectable()
    export class BackupMonitoringService {
      private readonly logger = new Logger(BackupMonitoringService.name);
      private readonly storage = new Storage();

      constructor(private readonly configService: ConfigService) {}

      @Cron('0 */4 * * *') // Every 4 hours
      async monitorBackupHealth(): Promise<void> {
        const environments = ['development', 'staging', 'uat'];

        for (const environment of environments) {
          try {
            const report = await this.generateHealthReport(environment);
            await this.processHealthReport(report);
          } catch (error) {
            this.logger.error(`Health check failed for ${environment}:`, error.stack);
          }
        }
      }

      private async generateHealthReport(environment: string): Promise<BackupHealthReport> {
        const bucketName = this.configService.get('backup.storage.bucket');
        const bucket = this.storage.bucket(bucketName);

        const [files] = await bucket.getFiles({
          prefix: `pre-production/${environment}/`,
          maxResults: 100,
        });

        if (files.length === 0) {
          return {
            environment,
            lastBackupTime: new Date(0),
            backupStatus: 'critical',
            backupSize: 0,
            retentionCompliance: false,
            issues: ['No backups found'],
          };
        }

        // Sort by creation time, newest first
        files.sort((a, b) => new Date(b.metadata.timeCreated).getTime() - new Date(a.metadata.timeCreated).getTime());

        const latestBackup = files[0];
        const lastBackupTime = new Date(latestBackup.metadata.timeCreated);
        const backupAge = Date.now() - lastBackupTime.getTime();

        const expectedInterval = this.getExpectedBackupInterval(environment);
        const issues: string[] = [];

        // Check backup freshness
        if (backupAge > expectedInterval * 2) {
          issues.push(`Backup is ${Math.round(backupAge / 3600000)} hours old`);
        }

        // Check retention compliance
        const retentionDays = this.getRetentionDays(environment);
        const oldBackups = files.filter((file) => {
          const fileAge = Date.now() - new Date(file.metadata.timeCreated).getTime();
          return fileAge > retentionDays * 24 * 3600 * 1000;
        });

        if (oldBackups.length > 0) {
          issues.push(`${oldBackups.length} backups exceed retention policy`);
        }

        const backupStatus = issues.length === 0 ? 'healthy' : issues.length === 1 ? 'warning' : 'critical';

        return {
          environment,
          lastBackupTime,
          backupStatus,
          backupSize: latestBackup.metadata.size,
          retentionCompliance: oldBackups.length === 0,
          issues,
        };
      }

      private async processHealthReport(report: BackupHealthReport): Promise<void> {
        this.logger.log(`Backup health report for ${report.environment}: ${report.backupStatus}`);

        if (report.backupStatus !== 'healthy') {
          await this.sendHealthAlert(report);
        }

        // Store metrics for monitoring dashboard
        await this.storeHealthMetrics(report);
      }
    }

    // scripts/environment-cli.ts

    import { Command } from 'commander';
    import { Logger } from '@nestjs/common';
    import { NestFactory } from '@nestjs/core';
    import { AppModule } from '../src/app.module';
    import { PreProductionBackupService, EnvironmentRefreshService, TestDataGeneratorService } from '../src/backup';

    const logger = new Logger('EnvironmentCLI');

    async function bootstrap() {
      const app = await NestFactory.createApplicationContext(AppModule);

      const backupService = app.get(PreProductionBackupService);
      const refreshService = app.get(EnvironmentRefreshService);
      const testDataService = app.get(TestDataGeneratorService);

      const program = new Command();

      program
        .name('env-manager')
        .description('Environment management CLI for pre-production environments')
        .version('1.0.0');

      // Backup commands
      program
        .command('backup')
        .description('Create backup for specified environment')
        .argument('<environment>', 'Environment to backup (development, staging, uat)')
        .option('--type <type>', 'Backup type', 'full')
        .action(async (environment, options) => {
          try {
            logger.log(`Creating backup for ${environment}...`);
            const job = await backupService.performEnvironmentBackup(environment);
            logger.log(`Backup completed: ${job.id}`);
          } catch (error) {
            logger.error('Backup failed:', error.message);
            process.exit(1);
          }
        });

      // Restore commands
      program
        .command('restore')
        .description('Restore environment from backup')
        .argument('<environment>', 'Target environment')
        .argument('<backup-id>', 'Backup ID to restore from')
        .option('--mask-data', 'Mask sensitive data during restore', false)
        .option('--preserve-users', 'Preserve existing users', false)
        .action(async (environment, backupId, options) => {
          try {
            logger.log(`Restoring ${environment} from ${backupId}...`);
            const job = await refreshService.refreshEnvironmentFromBackup(environment, backupId, {
              maskSensitiveData: options.maskData,
              preserveUsers: options.preserveUsers,
            });
            logger.log(`Restore completed: ${job.id}`);
          } catch (error) {
            logger.error('Restore failed:', error.message);
            process.exit(1);
          }
        });

      // Refresh commands
      program
        .command('refresh')
        .description('Refresh environment with fresh production data')
        .argument('<environment>', 'Target environment')
        .option('--mask-data', 'Mask sensitive data', true)
        .option('--preserve-users', 'Preserve existing users', true)
        .action(async (environment, options) => {
          try {
            logger.log(`Refreshing ${environment} with production data...`);
            const job = await refreshService.refreshEnvironmentFromProduction(environment, {
              maskSensitiveData: options.maskData,
              preserveUsers: options.preserveUsers,
            });
            logger.log(`Refresh completed: ${job.id}`);
          } catch (error) {
            logger.error('Refresh failed:', error.message);
            process.exit(1);
          }
        });

      // Test data commands
      program
        .command('generate-data')
        .description('Generate test data for environment')
        .argument('<environment>', 'Target environment')
        .option('--users <count>', 'Number of users to generate', '1000')
        .option('--products <count>', 'Number of products to generate', '500')
        .option('--orders <count>', 'Number of orders to generate', '2000')
        .option('--preserve-existing', 'Preserve existing data', false)
        .action(async (environment, options) => {
          try {
            logger.log(`Generating test data for ${environment}...`);
            await testDataService.generateTestData({
              environment,
              userCount: parseInt(options.users),
              productCount: parseInt(options.products),
              orderCount: parseInt(options.orders),
              preserveExistingData: options.preserveExisting,
              useRealisticData: true,
            });
            logger.log('Test data generation completed');
          } catch (error) {
            logger.error('Test data generation failed:', error.message);
            process.exit(1);
          }
        });

      // List commands
      program
        .command('list-backups')
        .description('List available backups')
        .argument('<environment>', 'Environment to list backups for')
        .option('--limit <count>', 'Number of backups to show', '10')
        .action(async (environment, options) => {
          try {
            // Implementation to list available backups
            logger.log(`Listing backups for ${environment}...`);
          } catch (error) {
            logger.error('Failed to list backups:', error.message);
            process.exit(1);
          }
        });

      await program.parseAsync();
      await app.close();
    }

    bootstrap().catch((error) => {
      console.error('CLI bootstrap failed:', error);
      process.exit(1);
    });

    ### Environment Corruption Recovery

    1. Identify the issue scope and impact
    2. Choose appropriate backup based on recovery point needs
    3. Notify stakeholders of estimated downtime
    4. Execute restore procedure
    5. Validate environment health
    6. Update stakeholders on completion

    ### Data Loss Incident Response

    1. Immediately stop all write operations to affected environment
    2. Assess data loss scope and timeline
    3. Identify most recent clean backup
    4. Coordinate with development team on lost work
    5. Execute restore with data recovery
    6. Implement measures to prevent recurrence
